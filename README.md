# 🚀 SiteLaunchers

Welcome to the official repository for **SiteLaunchers**. This site is built to host and showcase high-quality Roblox scripts with a clean, modern UI.

## 🛠️ Tech Stack
- **HTML5 / CSS3**
- **Tailwind CSS** (via CDN)
- **JavaScript** (Clipboard API)
- **Netlify** (Deployment)

## 📁 Repository Structure
- `index.html`: The main website interface.
- `scripts/`: (Create this folder) To host your `.lua` script files.

## 🚀 Deployment
This site is designed to be hosted on **Netlify**. Simply connect this GitHub repository to your Netlify account for automatic deployments.
